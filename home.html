Computer Programming

By: Darren

What Is Computer Programming?

Computer programming is the process of writing code to command a computer to run specific actions.

How Is Computer Programming Used Today?

  1. Website Design
    • Computer programming can make the parts of websites we see.
    • Computer programming languages such as HTML, CSS, and Javascript create the visual part of the website (the part you are seeing right now).
    • Some examples of this include the front page of Google Classroom, the designs you make on Canva, and the page you are on right now.
  2. Store Data
    • Computer programming can make parts of websites we can not see.
    • Computer programming languages such as Python, Java, and C# can create programs that store data.
    • Some examples of this include Google going through millions of websites to find an answer to your search result, Amazon finding your order and sending it to warehouses, and Youtube giving you video recommendations based off the previous videos you watched.
  3. Run Programs To Solve Problems
    • Computer programming can help you solve everyday problems.
    • You can create a program to roll a die, create a password, or solve a Ceasar CipherA cipher where the alphabet in the plain text is shifted by a number down the alphabet. For example, if the cipher is 4, A=E, B=F, C=G, etc..
